DRISHT - AN APPLIATION FOR VISUALY IMPAIRED

It's never crowded along the extra mile! Our project Drisht - an app for visually impaired, which allows the non sighted to experience features just like in a normal smartphone, has won 1st Runner Up Prize at IET MPSTME on campus! It has been an honour to get an awesome response by none other than Bhavesh Bhatia founder of Sunrise Candles, based in Mahabaleshwar, Maharashtra, India. Sunrise Candles is a candle making company run majorly by visually disabled people. Technology doesn't discriminate among humans, nor it has limitations. Its the people who do, and it's time treat the differently abled not unusually. This application allows the visually impaired to learn and grow using smart phones in a different way! It has been an amazing journey to work on this project for the betterment of the society!
